bullet = CreateProjectile('bullet', 12, 2)

end

CreateProjectile就是创建一个子弹，跟上的俩参数

第1个是使用图片

第2个是x坐标，第3个是y坐标

 

 

Update函数是必不可少的，否则就会报错

 

接下来讲update函数

这个函数就是指每帧做什么事情

 

xxxxxxxxxx
i=0
function Update()
    bullet = CreateProjectile('bullet', 12,i)
    i=i+1
end

这个就可以演示

最开始设置i=0

然后每帧创建一个子弹，y值为i，顺便把i`加1

 

 

然后是移动子弹

Move

 

 

xxxxxxxxxx
bullet = CreateProjectile('bullet', 12,1)
function Update()
   bullet.Move(0,1)
end

 

这样就可以看到这个子弹*神的上升（雾）了

 

多个子弹

你可能会想这么写

xxxxxxxxxx
function Update()
    bullet = CreateProjectile('bullet', 12,1)
    bullet.Move(0,1)
end

但是，木大！

因为每一个只上升了1像素，为什么呢，因为它每次更新的时候都刷掉了以前那个bullet的控制权（？）

正确写法

​x
bullets={}
function Update()
    local bullet = CreateProjectile('bullet', 12,1)
    table.insert(bullets, bullet)


    for i=1,#bullets do
        bullets[i].Move(0,1)
    end
end

那么这是什么意思呢？

非常简单，先创建了一个表，名叫bullets(注意加了s)，然后每次循环创建一个bullet变量，并把它插入到这个表里（因为我加了local 所以每次创建的bullet变量都是不同的），最后根据表长度分别给每给子弹move1

所以每一次生成要随机啊

j=-100

end

math.random就是生成随机数的，这里面是生成1到5的数字，如果=1的话就生成子弹

速度随机

xxxxxxxxxx
bullets[i].Move(0,math.random(1,5))

上面错误做法

下面正确做法

xxxxxxxxxx
j=-100
addj=1
spawn=0
bullets={}
function Update()
    if spawn==30 then
        local bullet = CreateProjectile('bullet', j,-60)
        table.insert(bullets, bullet)
        bullet.SetVar("speed",math.random(1,5))
        spawn=0
    end
    spawn=spawn+1
    
    if j>=50 then
        addj=math.random(-3,-0.1)
    elseif j<=-101 then
        addj=math.random(0.1,3)
    end
    j=j+addj
    for i=1,#bullets do
        bullets[i].Move(0,bullets[i].GetVar("speed"))
    end
end

其中，SetVar和GetVar分别是创建一个属性（简单理解）其中speed就是一个，把每个子弹初始化时都设置一个speed属性为随机（SetVar），每次移动的时候就会用到（GetVar）
